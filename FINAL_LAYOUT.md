# 📐 Финальная схема расположения элементов

## Динамическая адаптация

Popup "Специальное предложение" автоматически меняет высоту в зависимости от наличия товаров в корзине.

---

---

## Анимация перехода

### При добавлении первого товара:
```
[🎁 Popup] 120px  →  плавно (0.3s)  →  [🎁 Popup] 200px
                     появляется [🛒]
```

### При удалении всех товаров:
```
[🎁 Popup] 200px  →  плавно (0.3s)  →  [🎁 Popup] 120px
                     исчезает [🛒]
```

## Минимальные расстояния

### Без корзины:
- **Popup ↔ Наверх:** 90px

### С корзиной:
- **Popup ↔ Корзина:** 100px
- **Корзина ↔ Наверх:** 70px
- **Popup ↔ Наверх:** 170px (суммарно)

## CSS & JavaScript

### CSS (базовая позиция):
```css
.discount-popup {
    bottom: 120px;  /* Начальная позиция */
    transition: all 0.3s ease;  /* Плавность */
}
```

### JavaScript (динамическое изменение):
```javascript
// При появлении товаров
discountPopup.style.bottom = '200px';

// При очистке корзины
discountPopup.style.bottom = '120px';
```

## Преимущества решения

✅ **Автоматическая адаптация** - не требует действий пользователя  
✅ **Плавная анимация** - приятный UX  
✅ **Оптимальное использование пространства** - popup близко когда возможно  
✅ **Нет перекрытий** - все элементы доступны всегда  
✅ **Логичная иерархия** - важные элементы не скрываются
