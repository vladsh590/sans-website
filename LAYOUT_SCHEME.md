# 🎨 Схема расположения элементов в правом нижнем углу

## Визуальная структура

```
                    Правый нижний угол →

┌─────────────────────────────────────────────┐
│                                             │
│                                             │
│                     ▲                       │
│                     │ 120px от низа         │
│              ┌──────────────┐               │
│              │   🎁 Попап   │               │ ← Popup со скидкой
│              │  со скидкой  │               │   (самый верхний)
│              └──────────────┘               │
│                     │                       │
│                     │ ~20-40px              │
│                     ▼                       │
│                    ┌───┐                    │
│                    │🛒9│                    │ ← Кнопка корзины
│                    └───┘                    │   (в середине)
│                     │                       │
│                     │ ~50px                 │
│                     ▼                       │
│                    ┌───┐                    │
│                    │ ↑ │                    │ ← Кнопка "наверх"
│                    └───┘                    │   (внизу)
│                     │                       │
│                     │ 30px                  │
│                     ▼                       │
└─────────────────────────────────────────────┘
                   Край экрана
```

## Детальное описание элементов

### 1. 🎁 Popup "Специальное предложение"
**Позиция:** `bottom: 120px; right: 20px;`
**Размер:** Адаптивный (зависит от контента)
**Z-index:** 1000
**Состояния:**
- Мини (свёрнутый)
- Полный (развёрнутый)

**Отступы:**
- От края справа: 20px
- От низа: 120px
- Расстояние до кнопки корзины: ~20-40px

### 2. 🛒 Кнопка корзины
**Позиция:** Динамическая (появляется при добавлении товаров)
**Размер:** 60x60px (круглая)
**Z-index:** 998
**Элементы:**
- Иконка корзины
- Красный badge с количеством

**Отступы:**
- От края справа: 20px
- От низа: ~80-100px (динамически)
- Расстояние до popup: ~20-40px
- Расстояние до кнопки наверх: ~50px

### 3. ↑ Кнопка "Наверх"
**Позиция:** `bottom: 30px; right: 30px;`
**Размер:** 50x50px (круглая)
**Z-index:** 999
**Появление:** При прокрутке вниз

**Отступы:**
- От края справа: 30px
- От низа: 30px
- Расстояние до кнопки корзины: ~50px

## Расчёт расстояний

```
Высота от низа экрана:
├─ 30px   ← Кнопка "наверх" (50px высота)
├─ 80px   ← Промежуток
├─ 80px   ← Кнопка корзины (60px высота)  
├─ 140px  ← Промежуток
└─ 120px  ← Popup (начало)
```

## Адаптивность

### Desktop (> 640px):
```css
.discount-popup { bottom: 120px; right: 20px; }
.cart-button { bottom: ~80-100px; right: 20px; }
.scroll-to-top { bottom: 30px; right: 30px; }
```

### Mobile (≤ 640px):
```css
.discount-popup { bottom: 110px; right: 16px; left: 16px; }
.cart-button { bottom: ~70-90px; right: 16px; }
.scroll-to-top { bottom: 30px; right: 30px; }
```

На мобильных popup растягивается на всю ширину (`left: 16px`).

## Взаимодействие элементов

### Сценарий 1: Появление popup
1. Пользователь заходит на страницу
2. Через 8-9 секунд появляется popup (120px от низа)
3. Кнопки корзины и "наверх" остаются доступны

### Сценарий 2: Добавление в корзину
1. Пользователь нажимает "В корзину"
2. Появляется кнопка корзины между popup и кнопкой "наверх"
3. Все элементы не перекрываются

### Сценарий 3: Прокрутка страницы
1. Пользователь прокручивает вниз
2. Появляется кнопка "наверх" (самая нижняя)
3. Popup и корзина остаются на своих местах

## Преимущества такой структуры

✅ Визуальная иерархия (сверху вниз: popup → корзина → наверх)  
✅ Нет перекрытия элементов  
✅ Все кнопки кликабельны  
✅ Popup заметен, но не мешает  
✅ Логичное расположение по важности  
✅ Адаптивность для мобильных устройств
