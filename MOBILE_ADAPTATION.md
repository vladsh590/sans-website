# 📱 Мобильная адаптация корзины и элементов

## Breakpoints (контрольные точки)

Адаптация реализована для трёх типов устройств:

### 1. 🖥️ Desktop (> 968px)
- Полноразмерные элементы
- Оптимальные отступы

### 2. 📱 Tablet (641px - 968px)
- Средние размеры элементов
- Адаптированные отступы

### 3. 📱 Mobile (≤ 640px)
- Компактные элементы
- Минимальные отступы для экономии места

## JavaScript адаптация

Позиция popup динамически меняется в зависимости от размера экрана:

```javascript
updateCartButton() {
    if (count > 0) {
        // С товарами в корзине
        if (window.innerWidth <= 640) {
            discountPopup.style.bottom = '220px';  // Mobile
        } else if (window.innerWidth <= 968) {
            discountPopup.style.bottom = '240px';  // Tablet
        } else {
            discountPopup.style.bottom = '200px';  // Desktop
        }
    } else {
        // Без товаров
        if (window.innerWidth <= 640) {
            discountPopup.style.bottom = '145px';  // Mobile
        } else if (window.innerWidth <= 968) {
            discountPopup.style.bottom = '160px';  // Tablet
        } else {
            discountPopup.style.bottom = '120px';  // Desktop
        }
    }
}
```

### Обработка поворота экрана

При изменении ориентации или размера окна:
```javascript
window.addEventListener('resize', () => {
    clearTimeout(resizeTimer);
    resizeTimer = setTimeout(() => {
        window.shoppingCart.updateCartButton();
    }, 250);
});
```

## Адаптация модального окна корзины

### Desktop
- Максимальная ширина: 800px
- Высота: до 90vh
- Padding: 24px 32px

### Tablet (≤ 968px)
- Ширина: 95% экрана
- Высота: до 95vh
- Padding: 18px 24px
- Карточки товаров: 70x70px

### Mobile (≤ 640px)
- Ширина: calc(100% - 16px)
- Высота: до 96vh
- Padding: 16px 20px
- Карточки товаров: 60x60px
- Заголовок: 1.3rem

## Особенности мобильной адаптации

### ✅ Popup:
- Растягивается на всю ширину (минус отступы)
- Уменьшенные шрифты
- Меньший размер иконки

### ✅ Кнопки:
- Пропорционально уменьшены
- Сохраняют кликабельность (min 44x44px)
- Оптимальные отступы от краёв

### ✅ Корзина:
- Адаптированная сетка товаров
- Компактные элементы управления
- Удобные кнопки количества

### ✅ Анимации:
- Плавные переходы сохранены
- Transition: 0.3s для всех устройств
