## 🎯 Как это работает

### Для пользователя:
1. Пользователь нажимает "В корзину" на любом товаре
2. Появляется плавающая кнопка корзины с количеством
3. Клик по кнопке открывает модальное окно с товарами
4. В корзине можно изменить количество или удалить товар
5. Кнопка "Оставить заявку" открывает форму заказа
6. После заполнения формы и отправки корзина очищается

### Для разработчика:
```javascript
// Пример структуры товара в корзине
{
  id: "scotch-48x990-transparent",
  name: "Скотч упаковочный 48мм*990м прозрачный",
  image: "../../images/products/scotch.jpg",
  quantity: 2
}
```

Данные хранятся в `localStorage` с ключом `shoppingCart`.
