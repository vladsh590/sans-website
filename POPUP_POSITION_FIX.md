# 📍 Изменение позиции popup со скидкой

## Проблема

Popup "Специальное предложение" находился в том же месте, что и кнопки корзины/наверх, перекрывая их или мешая взаимодействию.

## Решение

Popup перемещён ВЫШЕ по вертикали, чтобы не мешать кнопкам внизу справа.

### Было:
```css
.discount-popup {
    bottom: 20px;  /* Слишком близко к кнопкам */
}
```

### Стало:
```css
.discount-popup {
    bottom: 120px;  /* Выше кнопок корзины и "наверх" */
}
```

## Визуальная схема расположения

```
┌────────────────────────────────────────┐
│                                        │
│                                        │
│                                        │
│                                        │
│                                        │
│                      [🎁 Popup]       │  ← bottom: 120px
│                          ↓ 100px      │
│                      [🛒 Корзина]     │  ← bottom: ~80-100px
│                      [↑ Наверх]       │  ← bottom: 30px
└────────────────────────────────────────┘
```

## Отступы для всех элементов

### Desktop (обычные экраны):
- **Popup:** `bottom: 120px` (выше всех)
- **Кнопка корзины:** `bottom: ~80-100px` (в середине)
- **Кнопка "наверх":** `bottom: 30px` (внизу)

### Mobile (экраны < 640px):
- **Popup:** `bottom: 110px` (выше всех)
- **Кнопка корзины:** `bottom: ~70-90px` (в середине)
- **Кнопка "наверх":** `bottom: 30px` (внизу)

## Преимущества такого расположения

✅ Popup не перекрывает кнопки  
✅ Все элементы визуально разделены  
✅ Пользователь может кликнуть на любой элемент  
✅ Popup остаётся заметным  
✅ Сохраняется иерархия важности элементов

## Расстояния между элементами

- От popup до кнопки корзины: **~20-40px**
- От кнопки корзины до кнопки наверх: **~50px**
- От кнопки наверх до края экрана: **30px**
