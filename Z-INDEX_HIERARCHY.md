# 📊 Иерархия наложения элементов (Z-Index)

## Визуальная схема

```
┌─────────────────────────────────────────┐
│  z-index: 1001 - Модальное окно корзины │  ← Самый верхний слой
├─────────────────────────────────────────┤
│  z-index: 1000 - Навигация (Header)     │
├─────────────────────────────────────────┤
│  z-index: 999  - Кнопка "Наверх"        │
├─────────────────────────────────────────┤
│  z-index: 998  - Кнопка корзины         │
├─────────────────────────────────────────┤
│  z-index: 900  - Popup со скидкой       │  ← Под кнопками
├─────────────────────────────────────────┤
│  z-index: 1    - Hero секция            │
├─────────────────────────────────────────┤
│  z-index: 0    - Основной контент       │  ← Самый нижний слой
└─────────────────────────────────────────┘
```

## Детали элементов

### 1. Модальное окно корзины (z-index: 1001)
- **Класс:** `.cart-modal`
- **Описание:** Всплывающее окно с товарами в корзине
- **Поведение:** Появляется при клике на кнопку корзины, перекрывает весь экран
- **Расположение:** `css/styles.css` строка ~2613

### 2. Навигация - Header (z-index: 1000)
- **Класс:** `.header`
- **Описание:** Верхнее меню навигации
- **Поведение:** Фиксированное, всегда видимо при прокрутке
- **Расположение:** `css/styles.css` строка ~80

### 3. Кнопка "Наверх" (z-index: 999)
- **Класс:** `.scroll-to-top`
- **Описание:** Круглая красная кнопка со стрелкой вверх
- **Поведение:** Появляется при прокрутке вниз, справа снизу
- **Расположение:** `css/styles.css` строка ~1562

### 4. Кнопка корзины (z-index: 998)
- **Класс:** `.cart-button`
- **Описание:** Плавающая красная кнопка с иконкой корзины и счётчиком
- **Поведение:** Появляется когда в корзине есть товары, справа снизу
- **Расположение:** `css/styles.css` строка ~2567

### 5. Popup со скидкой (z-index: 900) ✅
- **Класс:** `.discount-popup`
- **Описание:** Всплывающее окно "Специальное предложение"
- **Поведение:** Появляется через 8-9 секунд, справа снизу
- **Расположение:** `css/styles.css` строка ~2290
- **Статус:** Исправлено - теперь не перекрывает кнопки

### 6. Hero секция (z-index: 1)
- **Класс:** `.hero-content`
- **Описание:** Контент главного баннера
- **Расположение:** `css/styles.css` строка ~224

## Правила использования z-index

### ✅ Правильно:
- Модальные окна: 1001+
- Фиксированные элементы UI: 1000
- Плавающие кнопки: 900-999
- Popup уведомления: 800-899
- Обычные элементы: 1-100

### ❌ Неправильно:
- Использовать z-index: 9999 (слишком высокий)
- Давать popup больший z-index чем кнопкам
- Не использовать z-index для перекрывающихся элементов

## Расположение кнопок на странице

```
┌────────────────────────────────────────┐
│                                        │
│                                        │
│                                        │
│                                        │
│                      [🎁 Popup]       │  ← z-index: 900
│                      [🛒 3]           │  ← z-index: 998
│                      [↑]              │  ← z-index: 999
└────────────────────────────────────────┘
   Правый нижний угол страницы
```

Теперь кнопки корзины и "наверх" всегда доступны для клика!
