# 🎯 Динамическое позиционирование popup

## Концепция

Popup "Специальное предложение" автоматически поднимается выше, когда появляется кнопка корзины с товарами.

## Логика работы

### Состояние 1: Корзина пустая
```
Позиция popup: bottom: 120px

┌────────────────────────────────┐
│                                │
│         [🎁 Popup]            │  ← 120px от низа
│             ↓ 90px            │
│         [↑ Наверх]            │  ← 30px от низа
└────────────────────────────────┘
```

### Состояние 2: В корзине есть товары
```
Позиция popup: bottom: 200px (поднимается на 80px)

┌────────────────────────────────┐
│         [🎁 Popup]            │  ← 200px от низа
│             ↓ 100px           │
│         [🛒 Корзина]          │  ← ~100px от низа
│             ↓ 70px            │
│         [↑ Наверх]            │  ← 30px от низа
└────────────────────────────────┘
```

## Реализация

### JavaScript (script.js)
```javascript
updateCartButton() {
    const count = this.getTotalCount();
    const cartButton = document.getElementById('cartButton');
    const discountPopup = document.querySelector('.discount-popup');
    
    if (count > 0) {
        cartButton.classList.add('visible');
        // Поднимаем popup выше
        if (discountPopup) {
            discountPopup.style.bottom = '200px';
        }
    } else {
        cartButton.classList.remove('visible');
        // Возвращаем на исходную позицию
        if (discountPopup) {
            discountPopup.style.bottom = '120px';
        }
    }
}
```

### CSS (styles.css)
```css
.discount-popup {
    position: fixed;
    bottom: 120px;  /* Начальная позиция */
    right: 20px;
    transition: all 0.3s ease;  /* Плавное перемещение */
}
```

## Поведение

### Когда добавляется первый товар:
1. Появляется кнопка корзины
2. Popup плавно поднимается на 80px вверх (0.3 секунды)
3. Все элементы остаются видимыми и доступными

### Когда корзина очищается:
1. Кнопка корзины исчезает
2. Popup плавно опускается обратно (0.3 секунды)
3. Возвращается к исходной позиции

## Преимущества

✅ **Адаптивность:** Popup автоматически подстраивается под наличие корзины  
✅ **Плавность:** Transition создаёт приятную анимацию  
✅ **Не перекрывается:** Всегда есть достаточно места для всех элементов  
✅ **Интуитивно:** Пользователь видит все элементы без конфликтов  
✅ **Оптимально:** Без корзины popup ближе, с корзиной - выше

## Расстояния

### Без корзины:
- Popup до кнопки "наверх": **90px**

### С корзиной:
- Popup до кнопки корзины: **~100px**
- Кнопка корзины до кнопки "наверх": **~70px**

## Альтернативные значения

Если нужно изменить высоту поднятия, отредактируйте в `script.js`:

```javascript
// Больше места (выше поднимается)
discountPopup.style.bottom = '250px';  // Вместо 200px

// Меньше места (ближе к корзине)
discountPopup.style.bottom = '180px';  // Вместо 200px
```
