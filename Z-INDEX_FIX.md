# 🔧 Исправление наложения элементов (z-index)

## Проблема

Popup "Специальное предложение" перекрывал кнопку корзины и кнопку "наверх", мешая пользователю взаимодействовать с ними.

## Решение

Изменена иерархия наложения элементов (z-index) в файле `css/styles.css`.

### Было:
```css
.discount-popup {
    z-index: 1000;  /* Перекрывал все кнопки */
}
.scroll-to-top {
    z-index: 999;
}
.cart-button {
    z-index: 998;
}
```

### Стало:
```css
.discount-popup {
    z-index: 900;   /* Под кнопками */
}
.scroll-to-top {
    z-index: 999;
}
.cart-button {
    z-index: 998;
}
```

## Иерархия z-index (от верхнего к нижнему)

1. **z-index: 1001** - Модальное окно корзины (`.cart-modal`)
2. **z-index: 1000** - Навигация (`.header`)
3. **z-index: 999** - Кнопка "наверх" (`.scroll-to-top`)
4. **z-index: 998** - Кнопка корзины (`.cart-button`)
5. **z-index: 900** - Popup со скидкой (`.discount-popup`) ✅ **ИСПРАВЛЕНО**

## Результат

✅ Popup больше не перекрывает кнопку корзины  
✅ Popup больше не перекрывает кнопку "наверх"  
✅ Все элементы правильно взаимодействуют друг с другом  
✅ Модальное окно корзины остается поверх всех элементов
