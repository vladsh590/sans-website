# 📱 Обновление расположения для мобильных устройств

## Изменение концепции

На мобильных устройствах (≤ 640px) popup "Специальное предложение" теперь расположен **СНИЗУ**, под кнопками корзины и "наверх".

---

## Новая схема для Mobile (≤ 640px)

### Состояние 1: Корзина пустая

```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│              ┌───┐                  │
│              │ ↑ │ 46px             │ ← 20px от низа
│              └───┘                  │
│                ↕                    │
│              60px                   │
│                ↕                    │
│  ╔═══════════════════════════════╗ │
│  ║    🎁 Специальное             ║ │ ← 80px от низа
│  ║       предложение!            ║ │
│  ╚═══════════════════════════════╝ │
└16px──────────────────────────────16px
```

**Порядок снизу вверх:**
1. Край экрана
2. Кнопка "наверх" (20px от низа)
3. Отступ 60px
4. **Popup (80px от низа)** ← ВНИЗУ

---

```

└16px──────────────────────────────16px
```

**Порядок снизу вверх:**
1. Край экрана
2. Кнопка "наверх" (20px от низа)
3. Отступ 60px
4. Кнопка корзины (80px от низа)
5. Отступ 70px
6. **Popup (150px от низа)** ← ВНИЗУ, под обеими кнопками

## Сравнение: Desktop vs Tablet vs Mobile

### Desktop (> 968px) - Popup СВЕРХУ

```
        [🎁 Popup]      ← 120px (сверху над кнопками)
            ↕ 100px
        [🛒 Корзина]    ← ~100px
            ↕ 50px
        [↑ Наверх]      ← 30px
```

### Tablet (641-968px) - Popup СВЕРХУ

```
    [🎁 Popup растянут] ← 160px (сверху над кнопками)
            ↕ 120px
        [🛒 Корзина]    ← 20px
            ↕ 70px
        [↑ Наверх]      ← 90px
```

### Mobile (≤ 640px) - Popup СНИЗУ ⭐

```
        [🛒 Корзина]    ← 80px
            ↕ 60px
        [↑ Наверх]      ← 20px
            ↕ 60px
    [🎁 Popup растянут] ← 80px (СНИЗУ под кнопками!)
```

## Преимущества нового расположения на мобильных

✅ **Popup не мешает просмотру контента**  
✅ **Кнопки всегда доступны сверху**  
✅ **Естественный порядок: действия сверху, реклама снизу**  
✅ **Легче закрыть popup** (ближе к краю)  
✅ **Удобно для большого пальца**

## Позиции по устройствам

### Без корзины

| Устройство | Popup позиция | Логика |
|------------|--------------|--------|
| **Desktop** | 120px | Сверху над кнопкой наверх |
| **Tablet** | 160px | Сверху над кнопкой наверх |
| **Mobile** | 80px | **СНИЗУ под кнопкой наверх** |

### С корзиной

| Устройство | Popup позиция | Логика |
|------------|--------------|--------|
| **Desktop** | 200px | Сверху над корзиной |
| **Tablet** | 240px | Сверху над корзиной |
| **Mobile** | 150px | **СНИЗУ под обеими кнопками** |

    bottom: 80px;  /* Над кнопкой наверх */

## JavaScript логика

```javascript
if (window.innerWidth <= 640) {
    // Mobile - popup СНИЗУ
    if (hasItemsInCart) {
        popup.style.bottom = '150px';  // Под обеими кнопками
    } else {
        popup.style.bottom = '80px';   // Под кнопкой наверх
    }
}
```

## Анимация на мобильных

### При добавлении товара:

```
Popup: 80px → 150px (поднимается на 70px вверх)
Корзина: появляется на 80px
```

### При удалении всех товаров:

```
Popup: 150px → 80px (опускается на 70px вниз)
Корзина: исчезает
```
