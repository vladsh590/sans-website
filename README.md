## 📝 Установка и использование

1. Разместите логотип компании в `images/logo.png`
2. Откройте `index.html` в браузере
3. Все стили находятся в `css/styles.css`
4. JavaScript функционал в `js/script.js`

## 🔧 Настройка

### Форма обратной связи
Для подключения формы к серверу:
1. Откройте `js/script.js`
2. Найдите секцию "Form submission"
3. Раскомментируйте код fetch и укажите URL вашего API

### Добавление карты
Для добавления интерактивной карты на страницу контактов:
1. Получите API ключ от Яндекс.Карты или Google Maps
2. Замените `.map-placeholder` на код карты в `pages/contacts.html`
3. Укажите координаты офиса

### Изменение цветов
Все цвета определены в CSS переменных в начале файла `css/styles.css`:
```css
:root {
    --primary-red: #C41E3A;
    --primary-blue: #1A2332;
    --gold: #B8860B;
    /* и т.д. */
}
```

## 🌐 Развертывание

Сайт готов к размещению на любом хостинге:
1. Загрузите все файлы на сервер

## ❓ Часто задаваемые вопросы

### Что такое .gitignore?
`.gitignore` - это файл для системы контроля версий Git. Он указывает, какие файлы НЕ нужно сохранять в репозитории (временные файлы, настройки редактора и т.д.). 

**Если вы не используете Git** - этот файл можно удалить, он не влияет на работу сайта.

### Зачем папка pages?
Папка `pages/` создана для организации структуры проекта. Все внутренние страницы находятся в одном месте, а главная страница `index.html` остается в корне (это стандартная практика).

### Как добавить новую страницу?
1. Создайте новый HTML файл в папке `pages/`
2. Скопируйте структуру из любой существующей страницы
3. Обновите ссылки в навигации всех страниц
4. Не забудьте про относительные пути (`../css/`, `../images/`)
