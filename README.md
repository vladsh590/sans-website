# Сайт компании САНС - Упаковочные материалы

Корпоративный сайт компании САНС (Пермь) - поставщика упаковочных материалов и складского оборудования.

## � Система корзины (ЗАВЕРШЕНО)

### Реализованный функционал:
- ✅ **151 кнопка "В корзину"** на 21 странице товаров
- ✅ Плавающая кнопка корзины с красным счетчиком
- ✅ Модальное окно корзины с управлением количеством
- ✅ Форма оформления заявки
- ✅ Сохранение корзины в LocalStorage
- ✅ Работает на всех страницах сайта

### Технические детали:
- JavaScript класс `ShoppingCart` в `js/script.js`
- Стили корзины в `css/styles.css`
- Иконка корзины `images/icons/shopping-cart.svg`
- Уникальные ID для каждого варианта товара
